PROJECTS

Below you can see some projects I've built with various technologies. The projects are displayed in sequential order to symbolize growth in development skills and application complexity.

AbCrunch

Fitness Center Company Website

AbCrunch is a Fitness Center that offers gym memberships and training. The front end makes use of bootstrap's grid system for a responsive user experience and Sass for clean versatile code. The application also features a gym near me page running asynchronous javascript code to render gym location data from server.

Vugo

Real Estate Company Website

Vugo is a real estate marketplace where users can filter through the latest home and agent listings. The User experience is developed with HTML, CSS, and Bootstrap. While the front end interface directives and functionality are pioneered by the AngularJs library using the MVC software design pattern.

CineBox

Film Platform

CineBox is an informative film platform. Browse and search through thousands of films, receives plot summaries, film cast, critic reviews and much more through the Movie Database API. CineBox is produced entirely in the Angular framework using several of its notable features, such as components, routing, Rxjs library, Typescript and the HttpClientModule .

Stockery

Investor Dashboard

Stockery is an interactive data driven Investor Dashboard that renders the latest stock market data into user friendly graphs and charts via Chart.Js and the alphavantage API.

This Project is still in development.

Skyrail Skate Shop

Ecommerce Website

Skyrail is an e-commerce website that sells skateboards, bikes, scooters, clothing and various accessories. The application is driven by active server pages executed in the Asp.net framework and a SQL server database enabling dynamic content delivery to users.

This Project is still in development.
